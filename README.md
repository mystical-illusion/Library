# Library Management System

## Description
This is a **menu-driven console Library Management System** built in Java using **Object-Oriented Programming (OOP)** principles (Encapsulation, Inheritance, Polymorphism).  
It allows admins and members to manage and access library resources efficiently.

---

## Features
- **Admin Role:** Add, remove, search, and view books.
- **Member Role:** Borrow, return, and view issued books.
- Designed `Book`, `Library`, `User`, and `Admin` classes to track records and availability.
- Interactive console-based menu for easy navigation.
